    ESIEABOT Mobile app

    About

    The Esieabot mobile application allows you to control the ESIEABOT robot remotely via Bluetooth. The robot also hosts a website to display camera feedback which is then used by the application. The application is coded in Kotlin and the robot OS in Python.
    ESIEABOT is a 2 motors robot with a RaspberryPi zero. The python program is running on RaspbianOS, it communicate by Bluetooth with the mobile app.
    The mobile app contain multiple fonctionnalities such as BluetoothAutoparing, wifi configuration, help guide and many more.

    Second prize at the Scientific and Technical Project fair on the ESIEA Laval campus. Reward for my project among 20 projects of 2nd year students.

    • Kotlin
    • Python
    • Android
    • RaspberryPi
